# Payrexx NodeJS SDK
Payrexx NodeJS/TS API Wrapper
https://github.com/WeslleyDeSouza/payrexx-nodejs

## 📦 Installation
<code>
   NPM: npm i --save payrexx@production
</code>

## 🔁 Transpile TS to JS
<code>
 gulp prod
</code>
